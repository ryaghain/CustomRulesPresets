# CustomRulesPresets

Adds the ability to create, save, and load custom rules presets.

If you would like to help out by reporting an issue with the mod, you may do so by creating an issue in the github repository, usually listed as "website" on this mod's thunderstore page.
Furthermore, if you can, please try to include the log file in your BepInEx directory. You may also include debug messages in the log by enabling them in this mod's config file before launching the game.
